# クラスベースのコンポーネント作成
```
// app/View/Components　と　resources/view/components　配下に作成
sail artisan make:component コンポーネント名

// app/View/Componentsのみに作成する場合
sail artisan make:component コンポーネント名 --inline
```

# マイグレーションとモデル同時作成
```
sail artisan make:model Model -m
```
